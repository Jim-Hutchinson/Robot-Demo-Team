Use Case #1: Successfully Recognize Product by QR

Step 1. Setup a db_config.py file with proper configurations in data_src.

Step 2. Use qrcode_generator to create a QR code with the same ID as the product you want.

Step 3. Run robot_main and select the desired product with matching ID

Step 4. Put the generated QR code in front of the camera

Step 5. If successful, the camera will close and print the product and its ID into the terminal.
