Use Case #2: Successfully log into order.html

Step 1. Clone and open Robot-Demo-Team in VS code (github link: https://github.com/Jim-Hutchinson/Robot-Demo-Team).

Step 2. Setup a db_config.php file with proper configurations in data_src/includes (if not already created).

Step 3. Start both Apache and MySql in the XAMPP control panel on your computer.

Step 4. Open the following link: http://localhost/Robot-Demo-Team/web_src/index.php

Step 5. Index will show input boxes for username and database_password

Step 6. Upon entering in a valid username and password (will be given), user will be redirected to order.html
